Fluxo OAuth2
  1. Clique em Entrar com o Sabiá — você é redirecionado para /login/.
  2. Na página de login, clique no botão Entrar com o Sabiá.
  3. O pacote gera um state, salva na sessão e redireciona para o servidor Sabiá.
  4. Após autenticar no Sabiá, o servidor retorna para /auth/sabia/callback/.
  5. O pacote valida o state, troca o código pelo token, busca o perfil e cria/atualiza o usuário Django.
  6. Você é redirecionado para /dashboard/ já autenticado.
